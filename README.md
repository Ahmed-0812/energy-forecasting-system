# ⚡ Smart Energy Demand Forecasting System

A Machine Learning web application that predicts energy consumption based on environmental and industrial factors.

---

## 📌 Features
- Real-time energy consumption prediction
- Interactive data visualizations
- Trend analysis using graphs
- Machine Learning-based forecasting

---

## 🛠️ Tech Stack
- Python
- Machine Learning (Random Forest)
- Streamlit
- Pandas & NumPy
- Matplotlib & Plotly

---

## 📊 Dataset
- Climate & Energy dataset (2020–2024)
- Includes temperature, CO₂ emission, humidity, industrial activity, etc.

---

## 🧠 How it Works
1. Data preprocessing & feature engineering  
2. Model training using Random Forest  
3. User inputs parameters through UI  
4. Model predicts energy consumption  
5. Results displayed with visualizations
